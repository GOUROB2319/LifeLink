# LifeLink - Professional Healthcare Platform

A life-saving healthcare platform connecting patients, blood/organ donors, doctors, and hospitals.

## Features

- **Emergency Blood Request System** - Real-time donor matching with critical/urgent/routine priority levels
- **Donor Dashboard** - Availability toggle, donation history, achievements, next eligibility countdown  
- **Medical Directory** - Search and find doctors and hospitals with verification status
- **Organ Donation Registry** - Education, myth-busting, and digital consent forms
- **Secure Authentication** - Google OAuth and phone authentication support
- **Professional Design** - Clean, trustworthy medical aesthetic following WCAG 2.1 AA standards

## Pages

### Core Pages
- `index.html` - Landing page with hero, statistics, testimonials
- `login.html` - Secure login with OAuth (Google/Phone)
- `emergency-request.html` - Emergency blood request dashboard with live donor matching
- `donor-dashboard.html` - Donor profile and activity dashboard

### Directory & Search
- `medical-directory.html` - Doctor and provider search
- `hospital-profile.html` - Hospital profile and resources  
- `organ-registry.html` - Organ donation registry and education

### User Management  
- `onboarding-1.html` - Multi-step onboarding (account setup)
- `settings.html` - User settings and privacy controls
- `donation-success.html` - Post-donation confirmation and impact summary

## Technology Stack

- **Frontend**: HTML5, TailwindCSS
- **Typography**: Inter font family
- **Icons**: Material Symbols Outlined
- **Images**: Hotlinked from Google (production-ready)
- **JavaScript**: Vanilla JS with Firebase UI simulation

## Design Philosophy

- **Clean & Professional**: "Doctor's White Coat" aesthetic
- **Emergency-Critical Actions**: Highly visible with pulse animations
- **Accessibility**: WCAG 2.1 AA compliant
- **Mobile-First**: Fully responsive across all devices
- **Zero Learning Curve**: Intuitive for elderly and less tech-savvy users

## Firebase Integration (Visual)

The app includes visual Firebase integration cues:
- Real-time counters (animated incrementing numbers)
- Live status indicators (green pulse dots)
- OAuth authentication buttons (Google, Phone)
- Dynamic donor matching displays
- Live availability toggles

Note: This is a high-fidelity prototype. Actual Firebase backend integration would require additional setup.
